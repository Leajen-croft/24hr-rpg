Become
a GM

Want to get involved as more than just a player? For a limited time only, we are accepting applications to become a GM for one of the sessions. You can pick any game system you like and will liaise with the existing GM’s to help make a cohesive experience for everyone involved.

Get in contact as soon as possible at kinoandhermes@gmail.com with the subject heading ‘I Want To Be A GM’ with the following details:

  • Name
  • Preferred game system
  • Maximum group size
  • Favourite tabletop moment
  • Number of campaigns run (or total number of hours)
